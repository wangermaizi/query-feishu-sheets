# 更新日志

本文件记录项目每个发布版本中面向使用者的重要变化。

## [v0.1.0] - 2026-07-17

### 新增

- 新增飞书需求自动执行 Skill，支持读取 Sheets 和 Bitable、分析排序并每轮选择一条需求。
- 新增代码修改前的分支确认，以及主 Agent 实施、三个只读 Agent 独立审查和统一修正流程。
- 新增本地处理状态、固定飞书群自动发送、结果哈希校验和敏感凭证保护机制。
- 新增离线安装包、SHA-256 校验文件及 GitHub Actions 自动构建和标签发布流程。
- 新增仓库发布 Skill，可自动整理更新日志、创建语义化版本并推送到 GitHub。

### 改进

- 流水线启动检测优先使用 GitHub CLI，并在本机未安装 CLI 时支持通过 GitHub REST API 查询公开仓库。
